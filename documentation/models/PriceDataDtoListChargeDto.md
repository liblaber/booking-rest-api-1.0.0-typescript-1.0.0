# PriceDataDtoListChargeDto

| :----------- | :--------------------------- | :------- | :-------------------------------------------------- |
| base         | number                       | ❌       | The base price. Does not include any extra charges. |
| book         | number                       | ❌       | The price that will be charged by Booking.com.      |
| total        | number                       | ❌       | The total price. Includes all extra charges.        |
